How to use

	<div class="container">
  	  <div class="row">
    	  	<div class="col s12 m4">Col1</div>
    	  	<div class="col s12 m4">Col2</div>
    	  	<div class="col s12 m4">Col3</div>
  	  </div>
	</div>

You can also use l* and xl* for large and xlarge. To use column offsets, use the class offset-s*, offset-m*, etc. For example, to offset a column 3 column lengths, use the class offset-s3. To offset a column 3 column lengths only for the medium media query use offset-m3. If you've included pull/pull classes, they work the same way as offsets you just need to substitute 'offset-' for 'pull-' or 'push-'. Note, rows do not need a container to work.
